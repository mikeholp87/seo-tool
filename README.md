# SEO Scout

A free SEO analyzer tool for new websites. Enter any website URL to get a comprehensive SEO audit with actionable recommendations.

## Features

- **8 Category Analysis**: Performance, Meta Tags, Headings, Images, Links, Technical, Content, Mobile
- **Overall Score**: Color-coded SEO score (0-100)
- **Actionable Recommendations**: Detailed fix steps for each issue
- **Expandable Results**: View detailed checks per category

## Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun dev

# Build for production
bun build

# Run linter
bun lint

# Run typecheck
bun typecheck
```

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS 4
